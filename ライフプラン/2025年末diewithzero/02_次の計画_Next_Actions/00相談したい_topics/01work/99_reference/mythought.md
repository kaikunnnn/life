# 悩み
- 転職するために使うサービスとしての位置づけや印象が強いが、本当はデザイナーにならずとも「デザイン」を世の中に実践する人が増えてほしい
- それは何かというと、デザインとはなにかというと、「自分のアイデアで世の中を良くして実行まで行える人を増やすこと」評論家ではなく実践者を増やす。わかりやすくいうと自分でサービスを作って夜に出して自分が課題に思っている小さなことでも変える人を増やすことです。ただ現状そうなってないし顧客の多くは転職したい人です。なので転職できるかどうかを第1に考えているので不安がそこにある、正直つまらない。自分が何をデザインできるかどうかを考えられる状態でデザインをしてないのでつまらないです。
- もっとつくる楽しさとか生み出すたのしさ、学ぶこと自体が面白いサービスにしたいです。なのでそう考えるとUI作るだけではなく、ビジネスやプロダクトグロースの話とかもろもろ教えたりしながら楽しんで自分のアイデアを世の中にぶつける人が増えてほしいです。
- なのでリブランドも本当は1番やりたいしメッセージを伝えていきたいし、なんならデザイナーになることが目的だとしても、ある種洗脳してしまってそういう自分が好きな人種に変えてしまいたい。
- あとはコンテンツをつくることがユーザーが必要だから（転職するには知らないとやばい内容、ユーザーの現状を見ると理解がされてないコンテンツの改善）やることが多く正直苦しい、仕方がないとはいえ。

### Q9. 「デザイン楽しい」が伝わるコンテンツ

**質問:**
「もっとデザイン楽しいっていうのが伝わるコンテンツにしたい」と言っていますが、**今のコンテンツで「楽しさ」が伝わっていない理由**は何だと思いますか？

- [○] 教育的すぎて堅い
- [○] 「正解」を教えすぎている
- [○] 制作の「過程」ではなく「結果」だけ見せている
- [○] あなた自身が楽しそうに見えない
- [ ] その他:

**👉 ここに答えを書く:**
まず自分が楽しくないです。教えているので。教えることは別に楽しいことではなく全く別物だと思います。
デザインを楽しくするには？それを楽しく伝えて解説になっていてそれを利用して視聴者がデザインできるようになるには？というBONOのコンテンツ作りのスタイルを考えたいです。アイデアをください。私のYouTubeの動画ごとのアナリティクスを与えたら良いですか？関係ないですか？

**具体的改善案:**
「デザイン楽しい！」が伝わるコンテンツに変えるには？

例:
- タイトルを変える: 「UIデザインの基礎」→「このUIデザイン、クソださいから直す」
- 企画を変える: 「正しいデザイン手順」→「3時間でゼロからアプリUI作ってみた【発狂】」
- トーンを変える: 「こうすべきです」→「これマジで気持ちいいから見て！」

**👉 ここに答えを書く:**
たしかに感情的になるのはかなり必要です。自分が面白いって思うもの、もしくは面白く見れそうなものを伝える姿勢がめっちゃ必要だと思う。
あとは解説メインではなく、自分が制作する過程をもっと見せるのが良い気がする。今は「本」的にコンテンツを無料版→有料版で出している気がする（たとえばYouTubeは無料版だけど解説のみ、実践のやり方やイメージは課金するBONOで見せる」などしている。無料から剥き出しで制作過程を公開して続き見て学びたかったらBONOで、にできないかな？分かって入るけどコンテンツの型がわからないから具体的に相談したいかもです
